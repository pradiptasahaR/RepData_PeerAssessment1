Count missing values

The strategy is to replace a missing value with the average for its interval

z <- merge(rawdata,byInterval,by.x="interval",by.y="int")
z$steps[is.na(z$steps)] <- z$nsteps[is.na(z$steps)]
adjdata <- z[,1:3]
